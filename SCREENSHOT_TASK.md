## مراحل اجرا

1. فایل را در مرورگر باز کن (Chrome یا Safari)
2. پنجره مرورگر را روی عرض **1440px** تنظیم کن
3. از هر بخش زیر یک اسکرین‌شات جداگانه بگیر و ذخیره کن:

| نام فایل خروجی | بخش |
|---|---|
| `01_navbar.png` | نوار ناوبری بالای صفحه |
| `02_hero.png` | Hero Section کامل |
| `03_services.png` | بخش خدمات (۶ کارت) |
| `04_projects.png` | بخش پروژه‌ها |
| `05_testimonials.png` | نوار نظرات |
| `06_about.png` | بخش درباره ما |
| `07_catalog.png` | بنر کاتالوگ |
| `08_consultation.png` | فرم درخواست مشاوره |
| `09_collaboration.png` | بخش همکاری با ما |
| `10_footer.png` | فوتر |
| `00_fullpage.png` | اسکرین‌شات کامل کل صفحه (full-page) |

4. همه فایل‌ها را در این پوشه ذخیره کن:

## نکات مهم

- قبل از اسکرین‌شات مطمئن شو فونت IRANSans بارگذاری شده (چند ثانیه صبر کن)
- دکمه شناور پایین-چپ در همه اسکرین‌شات‌ها نباید دیده شود (می‌توانی آن را مخفی کنی)
- برای اسکرین‌شات full-page از ابزار full-page screenshot مرورگر استفاده کن
